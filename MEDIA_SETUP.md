# Media Files Setup Instructions

To complete your portfolio setup, please add the following files to the `public/media/` folder:

## Required Files:

1. **Profile Image**: `profile.jpg` or `whatsapp-image.jpg`
   - This will be displayed as your profile picture in the hero section
   - Recommended size: 400x400px or larger (square aspect ratio)
   - Supported formats: JPG, PNG

2. **Resume PDF**: `himateja-fullstack-dev.pdf`
   - This will be used for the resume download and preview
   - Make sure it's a valid PDF file
   - Recommended: Single page or multi-page resume

## How to Add Files:

1. Locate the `public/media/` folder in your project
2. Copy your WhatsApp profile image and rename it to `profile.jpg` (or keep as `whatsapp-image.jpg`)
3. Copy your resume PDF and rename it to `himateja-fullstack-dev.pdf`
4. Restart your development server (`npm start`) to see the changes

## File Structure:

```
public/
  media/
    profile.jpg          (or whatsapp-image.jpg)
    himateja-fullstack-dev.pdf
```

Once you've added these files, your portfolio will display your photo and allow resume downloads!
